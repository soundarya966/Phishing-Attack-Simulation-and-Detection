# Phishing Attack Simulation & Detection

The objective of this task is to understand how phishing attacks work, simulate a phishing campaign in a controlled environment, analyze user responses, identify red flags, and learn prevention techniques.

---

## Tools Used
- GoPhish (open-source phishing simulation framework)
- Manual phishing email templates
- Local HTML landing page
- Kali Linux / Linux environment

---

## Task Objectives
- Understand phishing attacks
- Create a fake phishing email template
- Design a phishing landing page
- Simulate a phishing campaign (test environment)
- Track and analyze responses
- Identify phishing red flags
- Learn phishing prevention methods
- Document the entire simulation

---

## Repository Contents
- `phishing_report.md` → Detailed phishing simulation report
- `email_template.txt` → Sample phishing email template
- `landing_page.html` → Fake login landing page (educational)
- `screenshots/` → Evidence of simulation and analysis

---

## ⚠️ Disclaimer
This project is created **strictly for educational and awareness purposes**.  
No real users were targeted, and no real credentials were collected.

## Final Outcome
✔ Improved understanding of social engineering  
✔ Awareness of phishing detection and prevention
